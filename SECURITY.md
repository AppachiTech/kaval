# Security Policy

## Reporting Security Issues

If you discover a security vulnerability in Kaval, please report it privately via email:

**madhu@appachi.tech**

**Do NOT open a public GitHub issue for security vulnerabilities.**

## What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment:** within 48 hours
- **Fix timeline:** depends on severity, typically within 7 days for critical issues

## Supported Versions

Only the latest release receives security updates.

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | No        |

## Scope

Kaval can terminate processes on the host system. Security concerns related to process killing, privilege escalation, or unexpected system behavior are especially relevant.
